# Guia de Instalacion y configuracion de Windows Terminal:

1. Instalaciones de Microsoft Store

    - Terminal
    - PowerShell

2. Configuracion de Windows Terminal

    Establecer PowerShell como predeterminado:
    1. Ctrl+,
    2. Inicio\Perfil Predeterminado -> PowerShell

    Instalacion de la fuente
    1. Acceder a la pagina web de NerdFonts -> www.nerdfonts.com/font-downloads
    2. Seleccionar una fuente y descargarla
    3. Descomprimir el archivo descargado e instalar con clic derecho

    Configuracion del tipo de fuente
    1. Ctrl+,
    2. PowerShell>Apariencia\Tipo de fuente -> Fuente instalada

3. Instalacion y configuracion de OhMyPosh

    1. Abrir la terminal
    2. Escribir el siguiente comando en la terminal -> winget install JanDeDobbeleer.OhMyPosh -s winget
    3. Escribir el siguiente comando en la terminal -> New-Item -Path $PROFILE -Type File -Force
    4. Acceder a la pagina web de ohmyposh -> ohmyposh.dev/docs/themes
    5. Seleccionar su tema favorito  y copiar el nombre del mismo
    6. Escribir el siguiente comando en la terminal editar el archivo de configuracion de PowerShell -> notepad $PROFILE
    7. Escribir en el archivo las siguientes lineas:
        oh-my-posh init pwsh --config "$env:POSH_THEMES_PATH\NombreTema.omp.json" | Invoke-Expression
        Import-Module -Name Terminal-Icons
    8. Cambiar el texto "NombreTema" por el nombre del tema que seleciono y guardar el archivo
    9. Escribir el siguiente comando en la terminal para recargar la configuracion -> . $PROFILE

4. Configuracion de la vision de archivos

    1. Escribir en la terminal el siguiente comando para instalar la configuracion -> Install-Module -Name Terminal-Icons -Repository PSGallery
    2. Escribir Y pulsar Enter para confirmar la instalacion
